Effective Date: May 16, 2025

This Privacy Policy explains how Digitronium Softworks ("we", "our", or "us") collects, uses, stores, and protects user data in connection with the Digitronium Discord bot ("Digitronium", "the Service").

1. Information We Collect

Digitronium may collect the following data when users interact with the bot:

  • Server IDs
  • Message content
  • Timestamps
  • IP addresses

All data is collected only when users issue commands directly to the bot.

2. Purpose of Data Collection

We collect this data for the following purposes:

  • Moderation functionality
  • User behavior tracking
  • Service debugging
  • Improving user experience through feedback analysis

3. Data Retention

Collected data is stored for a period of up to one (1) month and then removed or anonymized unless required longer for debugging or compliance purposes.

4. Data Usage with Third Parties

We share limited data with Groq AI for the purpose of delivering AI-powered features. We do not train AI models with user data.

5. Data Storage and Security

User data is stored on servers hosted by GitHub. At this time, data is not encrypted. Access to the data is restricted to authorized members of the Digitronium Softworks team only.
